Инструкция для iOS

Раскройте
  1. Установите приложение FoXray
  2. Откройте приложение
    Cогласитесь с правилами
    Отключите пуши (не понятно зачем они нужны)
  3. Установите ip файлы Нажмите 1 и подтвердите открытие в FoXray
  4. Установите domain файлы Нажмите 2 и подтвердите открытие в FoXray
  5. Установите настройки роутинга Нажмите 3 и подтвердите открытие в FoXray
  6. Установите настройки VPN Нажмите 4 и подтвердите открытие в FoXray
  7. Включите правила редиректа
    1. Нажмите на Xray setting
    2. Нажмите на кружок Ru-direct и на Save
  8. Установка и первый запуск VPN
    1. Запустите VPN
    2. Подтвердите установку VPN
  9. Дальнейший запуск и остановку VPN можно производить в меню настроек iOS
    Запустить VPN
    Остановить VPN

Инструкция для Android

Раскройте
  1. Установите приложение v2rayNG
  2. Скачайте файл с конфигурацией
  3. Загрузка файла конфигурации в VPN
    1. Запустите приложение v2rayNG и вызовите боковое меню
    2. Выберите О приложении
    3. Выберите Восстановление конфигурации
    4. Разрешите доступ к файлам (возможно окошко будет выглядеть по-другому)
    5. Выберите файл скаченный ранее (возможно окошко будет выглядеть по-другому)
    6. Посмотрите на сообщение с успешной загрузкой конфигурации
  4. Загрузка ip и domain файлов
    1. Вызовите боковое меню
    2. Выберите пунт меню Файлы ресурсов
    3. Нажмите на значок загрузки в вверхней части экрана
    4. Проверьте, что у двух нижних файлов появилась дополнительная информация
  5. Установка и первый запуск VPN
    1. Запустите VPN
    2. Разрешите установку VPN (возможно окошко будет выглядеть по-другому)
  6. Дальнейший запуск и остановку VPN необходимо выполнять из приложения v2rayNG
    Запустить VPN
    Остановить VPN
